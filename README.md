# KNX Dimmer

W = Single
CW = Cold/Warm White
RGB = RGB
RGBW = RGB+White
RGBCW = RGB+Cold/Warm White

Possible outputs:
3X  RGBCW
or
4x  RGBW
or
5X  RGB
or
8X  CW
or
16X W
